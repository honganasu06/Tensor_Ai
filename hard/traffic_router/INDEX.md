## üí° Key Concepts

### Cost Function
```
Total Cost = Distance + (Distance √ó Traffic) + Toll
```

### Algorithm Flow
```
Initialize ‚Üí Priority Queue ‚Üí Loop:
  Pop Best ‚Üí Check Visited ‚Üí Mark ‚Üí Explore Neighbors ‚Üí Loop
  Result when destination reached first
```

### Constraint Handling
```
Hard Constraints (must satisfy): Fuel
Soft Constraints (preferences): Toll budget
```

### State Tracking
```
(cost, node, route, distance, toll, fuel, fuel_stops)
```

## üêõ The 5 Bugs Explained

1. **Visited Check Timing** - Check AFTER should be BEFORE
2. **Fuel Calculation** - Using 0.9√ó instead of exact value
3. **Cost Ordering** - Toll in wrong place in calculation
4. **Constraint Order** - Checking toll before fuel validation
5. **Fuel Stops Tracking** - Inconsistent list management

**Find them in:** `buggy_router.py`  
**Understand them:** `BUG_ANALYSIS.md`  
**Fix them:** Yourself! (learning exercise)
